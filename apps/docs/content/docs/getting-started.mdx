# Getting Started

Welcome to AgileFlow! This guide will help you get started with AgileFlow in your development workflow.

<Callout type="info">
  AgileFlow is an AI-driven agile development system designed to work seamlessly with Claude Code, Cursor, and Windsurf IDEs.
</Callout>

## What is AgileFlow?

AgileFlow brings powerful agile project management capabilities directly into your IDE. With **41 commands**, **26 specialized agents**, and **23 skills**, AgileFlow automates and streamlines your entire development workflow.

### Key Features

- **Epic Planning**: Break down large features into manageable stories with dependencies
- **Sprint Management**: Plan sprints with velocity tracking and capacity planning
- **Architecture Decisions**: Document technical decisions with ADRs
- **CI/CD Integration**: Automate testing and deployment workflows
- **Documentation Sync**: Keep docs in sync with code changes
- **Multi-IDE Support**: Works with Claude Code, Cursor, and Windsurf

## Prerequisites

Before installing AgileFlow, ensure you have:

- **Node.js**: Version 18 or higher
- **npm**: Version 9 or higher
- One of the supported IDEs:
  - Claude Code
  - Cursor
  - Windsurf
