# Welcome to AgileFlow

AgileFlow is an AI-driven agile development system that brings powerful automation to your workflow with **41 commands**, **26 agents**, and **23 skills** across multiple IDEs.

## Quick Start

Install AgileFlow with a single command:

```bash
npx @xloxn69/agileflow install
```

## Features

- **Epic Planning**: Break down large features into manageable stories
- **Sprint Management**: Plan and track sprint progress
- **Architecture Decisions**: Document technical decisions with ADRs
- **CI/CD Integration**: Automate testing and deployment workflows
- **Documentation Sync**: Keep docs in sync with code changes
- **Multi-IDE Support**: Works with Claude Code, Cursor, and Windsurf
