# AgileFlow

**AI-driven agile development for Claude Code, Cursor, Windsurf, and more.** AgileFlow combines Scrum/Kanban planning, ADRs, and docs-as-code workflows into one framework-agnostic system.

## Install

```bash
npm install --save-dev agileflow
```

To skip auto-setup:

```bash
AGILEFLOW_SKIP_INSTALL=true npm install --save-dev agileflow
```

## CLI

```bash
npx agileflow setup
npx agileflow status
npx agileflow update
npx agileflow doctor
npx agileflow uninstall
```

## In Your IDE

After setup:

- `/AgileFlow:setup` - Scaffold docs/templates/CI (optional)
- `/AgileFlow:help` - View commands and usage
